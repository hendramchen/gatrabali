Gatrabali

Pengobatan Tradisional Bali

Pendahuluan

Website adalah media digital yang wajib ada pada suatu bisnis atau organisasi saat ini. Mengingat banyak masyarakat yang ketergantungan dengan gadget (gawai) akhir-akhir ini. Secara garis besar website terdiri dari 2 jenis sesuai fungsi nya. Website hanya menampilkan informasi, dan web app (website aplikasi) atau yang bisa berinteraksi dengan pengunjung, seperti fitur booking, sistem admin.

Web Gatrabali adalah salah satu jenis website aplikasi yang memberikan layanan bertanya kepada pengusada, konsultasi, dan fitur admin untuk mengelola konten. Dalam web ini juga akan ada 3 jenis user berbeda, yaitu, admin, pengusada, dan visitor. Mereka memiliki hak akses yang berbeda sesuai jenis user nya.

Dokumen ini akan menjelaskan cara penggunaan website Gatrabali baik dari sisi admin atau dari sisi depan (pengunjung dan pangusada).

Sistem Administrator (Admin)

Untuk dapat login ke sistem admin, user bisa akses url atau alamat berikut di browser: https://gatrabali.org/admin/login

Untuk dapat login ke halaman admin, user diwajibkan untuk menggunakan domain gatrabali. Seperti admin@gatrabali.org, hello@gatrabali.org , atau yang berakhiran @gatrabali.org. Selain domain itu, user tidak bisa login ke halaman admin.

Sistem Admin memiliki 8 menu yang terletak disebelah kiri, yaitu: Dashboard, Tentang, Artikel, Konsultasi, Kabupaten, Pengusada, Pertanyaan, Pengguna.

Dashboard

Setelah login berhasil, maka user akan dihadapkan dengan halaman dashboard. Seperti gambar berikut.

Ada beberapa panel widget di dashboard, yaitu grafik untuk menghitung jumlah user bertanya dan jawaban, jumlah user di masing-masing kabupaten.

Tentang

Pada halaman Tentang, pengguna bisa edit informasi yang ingin di tampilkan di bagian sistem depan. Pengguna hanya bisa edit Tentang yang sudah ada, meski pengguna bisa tambah baris informasi baru, namun yang dipakai hanyalah informasi Tentang yang pertama.

Artikel

Seperti kebanyakan web penyedia jasa, fitur artikel sangat penting untuk mempromosikan website, atau memberikan informasi yang relevan terkait apa jasa atau layanan yang kita berikan ke klien atau pengunjung website. Perlu diingat, admin atau copywriter harus rajin-rajin menulis artikel demi meningkatkan rating website di pencarian google, minimal seminggu sekali posting.

Konsultasi

Daftar konsultasi pada bagian admin digunakan untuk melihat klien atau pengunjung yang telah mengirim pesan konsultasi. Pihak admin bisa edit konsultasi dari status pending ke status done.

Kabupaten

Daftar kabupaten adalah master data untuk pengisian alamat pada data pengguna. Jadi data-data kabupaten sudah tersedia dan tinggal dipakai di halaman user. Jika setiap user yang registrasi melengkapi data-data nya seperti kabupaten dan kecamatan, kita bisa tau berpa jumlah user pada kabupaten Buleleng, Badung, dll.

Pengusada

Halaman pengusada berisi daftar pengusada dengan berbagai kategori yang ada. Tidak seperti user (visitor), pengusada didaftarkan langsung oleh tim admin, karena butuh verifikasi data. Sebelum menambah data pengusada, admin harus membuat user baru di halaman Pengguna dengan role user “pengusada”.

Pertanyaan

Daftar pertanyaan datang dari pengunjung-pengunjung yang sudah terdaftar. Disini admin atau tim admin bisa merespon pertanyaan dari pengunjung. Namun, disisi lain, pengusada bisa login dan balas pertanyaan lewat sistem depan.

Pengguna

Daftar pengguna terdiri dari pengunjung, pengusada dan admin. Admin user bisa menambahkan pengguna baru dan memberikan role atau tipe user itu sendiri, tipe admin, pengusada atau visitor.

Sistem Depan (Pengunjung & Pengusada)

Login

Saat user mengakses alamat website Gatrabali di: https://gatrabali.org/ , user bisa menemukan link untuk halaman login, yaitu menu Masuk di pojok kanan atas. Setelah pengunjung memasukkan email dan password dengan benar, dan login berhasil, maka pengunjung bisa mengakses penuh web Gatrabali, yaitu bisa mengajukan pertanyaan di halaman bertanya, mengajukan konsultasi di halaman konsultasi, serta memperbaharui profile.

Registrasi

Halaman Registrasi bisa ditemui dengan mengklik link “registrasi” pada halaman login. Pengunjung hanya butuh mengentry nama lengkap, email yang aktif, dan password yang aman.

Verifikasi Email

Verifikasi email penting dilakukan agar proses konsultasi atau buat pertanyaan bisa dilakukan. Untuk melakukan verifikasi email, user bisa ikuti langkah-lankah di video ini.

Lupa Password

Untuk halaman lupa password, user bisa klik link “lupa password” pada halaman login. Ketika user lupa password, maka user bisa mengubah password dengan menginput email terlebih dahulu, kemudian klik tombol “Email Password Reset Link”. Sistem akan mengirimkan link untuk reset password ke email aktif user. Misal sistem akan mengirimkan link ke email nama@gmail.com

Profile

Pada halaman profile, user bisa melengkapi data diri pengguna. Ini penting untuk proses konsultasi bisa lebih lancar.

Beranda

Halaman Utama atau Beranda, berisi informasi terkait website ini memberikan layanan apa saja ke pengunjung. Secara sederhana, halaman home berisi, header, layanan kami, dan beberapa artikel.

Info

Halaman Info adalah halaman yang berisi artikel-artikel yang ditulis oleh admin. Artikel ini berguna untuk mengedukasi atau memberikan informasi ke pengunjung seputar kesehatan. Perlu ditekankan, untuk membuat website kita lebih dikenal google, maka setidaknya ada satu artikel yang dipublish setiap minggu nya.

Bertanya

Fitur Bertanya adalah salah satu fitur dari Gatrabali. Dimana pengguna bisa bertanya seputar kesehatan. Sebisa mungkin admin atau pengusada bisa membalas pertanyaan tersebut. Hanya pengguna yang terdaftar yang bisa mengajukan pertanyaan. Pengguna yang belum mendaftar hanya bisa melihat daftar pertanyaan saja.

Pengusada

Halaman Pengusada menampilkan daftar pengusada dengan kategori tertentu. Pengguna juga bisa melihat detail dari pengusada dengan mengklik nama dari pengusada tersebut.

Konsultasi

Pada halaman konsultasi, pengguna yang telah terdaftar bisa melakukan konsultasi dengan mengisi formulir yang telah disediakan.

Tentang

Halaman tentang memuat informasi terkait siapa pendiri website Gatrabali, dan apa visi misi dari website Gatrabali.
